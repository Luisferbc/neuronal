# 🧠 Streamlit App - Red Neuronal vs Regresión Lineal (Versión Corregida)

Aplicación interactiva que compara una red neuronal (MLPRegressor) con regresión lineal.

## 🚀 Ejecutar

pip install -r requirements.txt
streamlit run app.py

## ⚠️ IMPORTANTE

En Streamlit Cloud asegúrate de configurar:
Main file path = app.py

## 🎯 Funcionalidades

- Entrenamiento dinámico
- Comparación de modelos
- Visualización gráfica
- Predicciones en tiempo real
