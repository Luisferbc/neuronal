# 🧠 Streamlit App - Red Neuronal vs Regresión Lineal

Aplicación interactiva que compara una red neuronal simple con un modelo de regresión lineal.

## 🚀 Ejecutar

pip install -r requirements.txt
streamlit run app.py

## 🎯 Funcionalidades

- Entrenamiento dinámico
- Comparación de modelos
- Visualización gráfica
- Predicciones en tiempo real
